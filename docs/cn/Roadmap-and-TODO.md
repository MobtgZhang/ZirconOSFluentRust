# 路线图与待办（Phase 0–15）

**English**: [../en/Roadmap-and-TODO.md](../en/Roadmap-and-TODO.md)

> **说明**：下表对应母版 [ideas/ZirconOS_NT10_Architecture.md](../../ideas/ZirconOS_NT10_Architecture.md) §25。**Status** 反映 **本仓库** 截至文档更新时的范围（以母版全栈内核为目标时，多数 Phase 仍为未开始）。

**对标范围**：文档与 syscall ABI 以 **NT 10.0.19045** 为固定基线；更高 NT10 构建不自动跟踪。**不追求**与任意 Windows 安装介质或更新构建的二进制兼容（clean-room 自有实现）。引导路径差异见 [Boot-paths.md](Boot-paths.md)。

**勘误（ideas/claude）**：[ideas/claude/content1.1.md](../../ideas/claude/content1.1.md) 部分段落假设较早树（如全盘 `PfnBump`、线性 VAD）；**以本表 Status 内链接的当前源码为准**。
